# Models trained with noisy genomes extend bacterial phenotype prediction into deep time
Predicting bacterial phenotypes from gene content using noise-robust machine learning, applied to ancestral genome reconstruction. Includes models for oxygen use, cell-wall type, optimal growth temperature, and sporulation, with inference back to the last bacterial common ancestor (LBCA).
